# WeatherApp

Descrição do Projeto
WeatherApp é um aplicativo web simples e funcional que permite aos usuários verificar a previsão do tempo de forma rápida. Ele combina a geolocalização automática para mostrar as condições climáticas atuais do usuário e um mecanismo de pesquisa para consultar o tempo em qualquer cidade ou país do mundo.

O projeto foi desenvolvido puramente com tecnologias front-end: HTML, CSS e JavaScript.

**Funcionalidades Principais**
Geolocalização Automática: Ao carregar a página, o aplicativo solicita a localização do usuário e exibe instantaneamente o clima, temperatura e condições da cidade atual.

Pesquisa Dinâmica: Permite que o usuário insira o nome de uma cidade ou país para obter a previsão do tempo correspondente.

Exibição de Dados: Mostra a temperatura, descrição do clima, umidade e outras informações relevantes do local pesquisado ou detectado.

Design Responsivo: (Assumindo que o CSS provê um layout agradável em diferentes tamanhos de tela.)

**Como Executar o Projeto**
Este projeto não requer um servidor de backend complexo. Basta seguir estes passos:

Clone ou Baixe: Baixe ou clone o repositório do projeto para o seu computador.

Abra o Navegador: Localize o arquivo index.html na pasta principal do projeto.

Execução: Clique duas vezes em index.html para abri-lo em seu navegador padrão (Chrome, Firefox, etc.).

**Nota: O navegador solicitará permissão para acessar sua localização na primeira vez que você carregar o aplicativo.**
